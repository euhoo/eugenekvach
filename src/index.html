Три инженерных решения, которые сделали систему предсказуемее.

Там, где поведение можно определить заранее, модель не вызывается. Там, где нужен reasoning, агент получает узкий набор инструментов, память и проверяемый сценарий.

  1. Fast-path до LLM

    Критичные триггеры вынесены до модели: быстрее и предсказуемее.

  2. Cache-first prompts

    Static/dynamic split сохраняет prefix cache и снижает стоимость.

  3. Scenario harness

    Диалоги, routing и tool use проверяются на сценариях из реальных ошибок.
