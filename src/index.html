02 · Главный кейс / Jarwis

Не обёртка над API, а production-система вокруг LLM.

Персональный Telegram-ассистент с инструментами, двухслойной памятью, обработкой текста, голоса и PDF — и детерминированными контурами для критичных сценариев.

request flow production · 2026
Текст · Голос · PDF
01 / route Coordinator определяет путь запроса
known trigger Deterministic
fast-path ~31 ms · 0 LLM calls
open request Specialized
agents reasoning · tool use
domain tools working memory ↔ long-term memory

Проблема → решение → результат

  1. 01

    Fast-path до LLM Критичные триггеры вынесены до модели: быстрее и предсказуемее.

  2. 02

    Cache-first prompts Static/dynamic split сохраняет prefix cache и снижает стоимость.

  3. 03

    Scenario harness Диалоги, routing и tool use проверяются на сценариях из реальных ошибок.

Разобрать инженерный кейс →
