Что держит платформу

  • Nx monorepoApps, libs, управляемые билды и codegen в одном дереве.
  • Shared UI-библиотекаОбщие компоненты и паттерны вместо копирования между приложениями.
  • Dynamic formsСложные процессы и валидация описываются схемой, а не собираются заново вручную.
  • Contract-firstТипизированные клиенты генерируются из контракта бэкенда.
  • Миграции частямиAngular 8 → 21 рядом с фичами, без паузы ради техники.
  • Качество на масштабеСтандарты, контракты и общая либа должны работать на гильдию, а не на одного человека.

Четыре инженерных решения

  • Fast-path до LLMСтатичные capability-триггеры вынесены в regex-путь до вызова модели: ~31 мс, ×80 и 0 LLM calls для известных сценариев.
  • Cache-first promptsStatic и dynamic части разделены: стабильный prefix повысил cache hit с 36% до 50% и снизил input cost примерно на 22%.
  • Scenario harnessTelegram-диалог, routing, tool calls, ответ и запись проверяются целиком на сценариях; harness нашёл 2 бага до релиза.
  • Memory lifecycleWorking memory живёт с TTL, а долгосрочные факты проходят recall, нормализацию и дедупликацию в PostgreSQL/pgvector.
